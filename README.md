## How to install
    1. In terminal   code ~/.bashrc
    2. In ./bashrc on line 4 -> export CHE_PROJECTS_ROOT=/workspace
    3. Exit bashrc file
    4. Exit terminal
    5. Reopen term
    6. Run in term -> sudo apt-get install subversion
    7. In term -> npm install -g opspark
    8. In term -> os install
